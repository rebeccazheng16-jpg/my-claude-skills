# 视频制作脚本格式规范

## 输出格式

```markdown
## 🎬 视频制作脚本

| 时间 | 印尼语口播 | 中文参考 | 画面 | B-Roll/素材 | 字幕 |
|------|---------|---------|------|-----------|------|
| 0:00-0:04 | [印尼语文案] | [中文文案] | HeyGen 正面中景 | — | 无 |
| 0:04-0:08 | [印尼语文案] | [中文文案] | HeyGen | — | 关键词加大 |
| 0:08-0:12 | [印尼语文案] | [中文文案] | 切 B-Roll | [素材描述] | — |
| ... | ... | ... | ... | ... | ... |
```

## 时间戳计算规则

- 印尼语口播速度：约 **150 词/分钟**（比中文略慢）
- 30 秒 ≈ 75 印尼语词
- 60 秒 ≈ 150 印尼语词
- 停顿点：每个段落转换处加 0.5-1 秒停顿

## 画面标注规范

| 标注 | 含义 |
|------|------|
| HeyGen 正面中景 | 数字人半身，标准口播画面 |
| HeyGen 正面近景 | 数字人面部特写，用于强调/金句 |
| 切 B-Roll | 切到实拍/素材画面 |
| 文字卡 | 纯文字画面（数据、对比等） |
| 产品特写 | 产品实物镜头 |

## B-Roll 素材描述规范

描述要**具体到可执行**，不要模糊：
- ✅ "滴水测试：海藻纤维 vs 棉质面料吸水速度对比，俯拍特写"
- ✅ "韩国皮肤科诊所门口标牌 + 室内环境，实拍或高清图片"
- ❌ "产品展示画面"
- ❌ "相关素材"
