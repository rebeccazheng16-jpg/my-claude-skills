# 三问代入法

> 来源：飞书妙记《直播电商业务规划与机制讨论》2026-03-04

## 金句原文

> "在产品介绍的过程中要带入你是谁、为什么买、为什么便宜这三个关键问题，而不是一味地为了一个产品去讲这个产品，只有完美解决这三个问题，才能做到极高的转化。"

> "做人设最基础的一点不是开播就说大家好，我是谁，而是要在只言片语中去代入我是谁。"

## 三问详解

### Q1: 你是谁

**错误做法**：
```
"大家好，我是 Lina，我做了十几年供应链，我去过几十家工厂..."
```

**正确做法**（只言片语代入）：
```
"我跑了十几年工厂，什么面料都见过。"
"我帮姐姐们选了上百个 SKU..."
"做供应链有个经验——别看面料新的时候怎么样，看它洗了二十次还剩什么。"
```

**Lina 可用的身份标签**：
- 供应链专家："我跑了十几年工厂"
- 产品规划师："我帮网红姐姐们做过上百个 SKU 的产品规划"
- 品牌操盘手："从品牌定位到包材设计，我都做过"
- 行业判断者："做了这么多年，我看面料只看一个标准"

### Q2: 为什么买

**错误做法**（功能罗列）：
```
"我们的产品有 7A 级抗菌、3 倍吸湿、2-3 度降温..."
```

**正确做法**（场景代入 + 扩展）：
```
"35 度不是最难受的，闷才是。坐一天办公室走出来，内裤是潮的。"
```

**场景扩展方法**：
| 原有场景 | 扩展思路 | 新增人群 |
|---------|---------|---------|
| 核心痛点 | 最明显的使用场景 | 核心受众 |
| 关联场景 | 同一产品的其他使用时机 | 相邻受众 |
| 意外场景 | "没想到也能用"的场景 | 新增受众 |

**参考案例**：筋膜枪不只卖健身→老人/小孩/上班族

### Q3: 为什么便宜

**错误做法**：
```
"超值优惠！限时特价！"
```

**正确做法**（真实可信 + 逻辑通顺）：
```
"我们代工厂同时在给 CK 和 Lululemon 做。同一家工厂。
CK 一条两三百。我们 6 条，600K。
差在哪？不是面料，不是品控。是品牌费、渠道费、广告费。"
```

**价格解释的合理理由**：
- 同厂代工（品质相同，品牌溢价不同）
- 品牌直供（没有中间商）
- 社交电商模式（口碑替代广告费）
- 韩国官网 vs 机制价（有实拍比价图支撑）

**禁止**：
- "限时""抢购""手慢无"
- 夸大折扣（"一折特价"）
- 无法验证的价格对比
