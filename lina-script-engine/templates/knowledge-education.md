# 认知科普型模板：行业揭秘 + 权威背书

> 来源：海藻纤维 V3 口播稿验证

## 核心理念

**用行业真相或材料/成分的专业知识建立认知壁垒，让观众觉得"只有她才能讲清楚这件事"。**

## 结构模板（5 段式）

### 1. 钩子：质疑或好奇

以"我也不信"的姿态开场，降低防备：

```
"说实话，我第一次听到也觉得离谱。"
"XX？做 XX？这不是智商税吗？"
"我第一反应——这靠谱吗？"
```

### 2. 权威背书故事

不直接讲产品，先讲这个技术/成分/材料的**专业出处**：

- 医学应用（"最早用在医用敷料里"）
- 大学研究（"青岛大学海洋纤维研究院"）
- 行业标准（"PDRN 含量 ≥5000ppm"）
- 历史起源（"1952 年意大利渔夫发现..."）

**用递进结构**（与品牌故事型相同）展现专业深度。

### 3. 行业对比揭秘

揭露行业"潜规则"，建立"我有你没有"的认知：

```
"市面上大部分 XX，其实就是..."
"这不是科技，这是化妆。"
"涂上去的东西，迟早洗掉。"
```

**核心句式**：真的 vs 假的、结构性 vs 涂层式、天性 vs 加工

### 4. 产品代入

从"行业真相"自然过渡到"我们怎么做"：

```
"我们用的是 XX，不是 XX。"
"数据不用我讲太多——[列具体数字]"
```

### 5. 场景扩展 + CTA

把产品和观众的**日常场景**连接：

```
"姐姐们在印尼待过就懂，35 度不是最难受的，闷才是。"
"坐一天办公室走出来，内裤是潮的。"
```

CTA 可以是关注型或轻度种草型（取决于是否附价格信息）。

## 三问侧重

| 问题 | 侧重 | 实现方式 |
|------|------|---------|
| 你是谁 | ✅ 重点 | 供应链经验自然代入 |
| 为什么买 | ✅ 重点 | 场景扩展 |
| 为什么便宜 | ⚠️ 可选 | 如附价格信息则加同厂锚点 |
