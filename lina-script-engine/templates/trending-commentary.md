# 热点评论型模板：热点 + 人设角度

## 核心理念

**借热点的流量，讲自己的观点。热点是钩子，人设是内核。**

## 结构模板（4 段式）

### 1. 热点切入

直接引用热点事件/新闻：

```
"最近看到一个数据——印尼女性创业者的平均存活率不到 2 年。"
"昨天一个新闻刷屏了——XX。"
```

**热点来源**：`indonesia-trending-news` skill 或用户指定。

### 2. Lina 的角度

不是复述热点，是给出**独特判断**：

```
"很多人在讨论 XX，但我觉得大家忽略了一件事——[洞察]"
"说实话，看到这个数据我不意外。因为 [基于 Lina 经验的分析]"
```

**必须用 Lina 的人设层次来解读**：供应链视角 / 女性创业者视角 / 品牌操盘手视角。

### 3. 可执行建议

不只是评论，要给观众**带走一个东西**：

```
"如果你现在正在 [热点相关场景]，我的建议是：第一...第二..."
"我自己的做法是 [具体做法]"
```

**规则**：建议要具体到可执行，不要"要有信心""要坚持"这种正确的废话。

### 4. 关注型 CTA

```
"这种话题我经常聊，关注我不错过。"
"你怎么看？评论区告诉我。"
```

## 三问侧重

| 问题 | 侧重 |
|------|------|
| 你是谁 | ⚠️ 隐性（通过解读角度展现） |
| 为什么买 | ❌ 不提 |
| 为什么便宜 | ❌ 不提 |

## 热点筛选标准

不是所有热点都值得做。筛选标准：

- ✅ 热点与 Lina 的专业领域相关（女性创业/供应链/印尼市场/品牌）
- ✅ Lina 能给出有真实经验支撑的独特观点
- ✅ 目标受众（印尼女性创业者/KOC）会关注这个热点
- ❌ 纯娱乐八卦
- ❌ 政治敏感
- ❌ Lina 没有相关经验，只能"正确地废话"
