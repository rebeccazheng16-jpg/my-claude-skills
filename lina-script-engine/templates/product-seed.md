# 产品种草型模板：三问全答 + 体感翻译

> 来源：koc-sales-script 方法论 + commerce-storyboard 体感翻译法 + 飞书会议三问代入法

## 核心理念

**三问必须全答。这是唯一需要同时回答"你是谁/为什么买/为什么便宜"的内容类型。**

## 结构模板（5 段式）

### 1. 痛点钩子（前 3 秒）

直击受众痛点，用**场景描述**而非问题句：

```
"35 度坐了一天办公室，走出来内裤是潮的。"
"敷完面膜还是拔干卡粉，你有没有怀疑过是面膜的问题？"
```

**禁止**："你有没有这样的困扰？"（太像广告）

### 2. 人设代入（只言片语）

用一句话展现专业身份，不自报家门：

```
"我帮姐姐们选了上百个 SKU，这种面料我一摸就知道行不行。"
"做了十几年供应链，我判断面料好不好只看一件事——洗了二十次还剩什么。"
```

### 3. 体感翻译（核心）

**从产品文件的"体感翻译表"直接提取**，用三步法：

| 步骤 | 方法 | 示例 |
|------|------|------|
| 参数→说人话 | 技术参数翻译成消费者语言 | "7A级抗菌" → "穿一天不闷不痒" |
| 功能→可拍动作 | 每个卖点对应画面 | 滴水测试、温度枪实测 |
| 形容词→具体感受 | 模糊描述变具体 | "舒适" → "穿上就忘了自己穿着东西" |

**规则**：消费者不需要查百度就能懂 = 翻译到位。

### 4. 价格锚点（必须）

产品种草型**必须回答"为什么便宜"**：

```
结构：[锚点价格] → [我们的价格] → [便宜的合理解释]

示例：
"CK 一条两三百。我们 6 条，600K。"
"韩国官网 110,000 韩币，约人民币 599。我们机制价 199，5 支。"
"不是我们便宜，是中间层少。"
```

**价格解释必须真实可信**：同厂代工 / 品牌直供 / 没有广告费。
**禁止**："超值""限时优惠""买到就是赚到"。

### 5. 促单 CTA

产品种草型的 CTA 要**引导行动**：

```
"链接在购物车，6 条装 600K。"
"点我主页，今天有活动。"
"私信我发'内裤'，我告诉你怎么买最划算。"
```

## 三问侧重

| 问题 | 侧重 | 实现方式 |
|------|------|---------|
| 你是谁 | ⚠️ 轻提 | 一句话代入 |
| 为什么买 | ✅ 核心 | 体感翻译 + 场景扩展 |
| 为什么便宜 | ✅ 核心 | 价格锚点 + 合理解释 |

## 质量基准

暂无验证范例。参考 `koc-sales-script` 的质量检查清单：
- □ 过嘴测试：念出来像聊天，不像念广告？
- □ 3 秒测试：前三秒有让人停下来的理由？
- □ 画面测试：每个卖点都有可拍画面？
- □ 促单测试：结尾让人选"买几个"而不是"考虑要不要买"？
