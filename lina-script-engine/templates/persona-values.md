# 人设价值观型模板：Lina 人设 + 趋势洞察

## 核心理念

**不讲产品，讲价值观。让观众认同 Lina 这个人，而不是某个产品。**

## 结构模板（4 段式）

### 1. 趋势观察

从一个**社会趋势或生活现象**切入：

```
"你有没有发现，这几年越来越多人开始往回收。"
"我身边很多朋友，三十五岁以后，衣柜里最常穿的全是基础款。"
"在印尼做了这么多年生意，我发现一件事——女性创业者最缺的不是钱，是被认真对待。"
```

### 2. 个人反思

Lina 对这个趋势的**真实思考**：

```
"我一直在想，为什么会这样？"
"后来我想明白了——[洞察]"
```

**必须符合 Lina 的价值观**：长期主义、赋能女性、诚信为本、务实落地。
**从 `lina-founder-style` 的核心价值观关键词中选择匹配项。**

### 3. 与品牌/事业的连接

从价值观**自然连接**到 Lina 在做的事：

```
"这就是我做图兰朵的原因——不是做时尚品牌，是做让你闭着眼睛抓一件就很舒服的品牌。"
"所以我每天在做的事，就是帮印尼的姐姐们找到最好的产品，用最低的成本。"
```

**规则**：连接要轻，点到为止。不要变成产品介绍。

### 4. 认同型 CTA

引导"我也是这样想的"的情感认同：

```
"如果你也觉得穿得舒服比穿得贵更重要，关注我。"
"如果你也开始在意自己的感受，我们慢慢聊。"
```

## 三问侧重

| 问题 | 侧重 |
|------|------|
| 你是谁 | ✅ 核心（人设本身就是"我是谁"的答案） |
| 为什么买 | ❌ 不提 |
| 为什么便宜 | ❌ 不提 |
