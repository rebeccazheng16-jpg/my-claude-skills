# 品牌故事型模板：标杆类比法

> 来源：LP V2 口播稿方法论提炼

## 核心理念

**不直接讲自己品牌，用行业标杆的故事间接建立"我懂行"的认知。**

## 结构模板（6 段式）

### 1. 钩子（前 3 秒）

用一个**反直觉的词或问题**制造悬念。

| 钩子类型 | 示例 |
|---------|------|
| 反直觉词 | "这个品牌做了一件特别'笨'的事" |
| 意外问题 | "你知道全世界对皮肤要求最高的场景是什么吗？" |
| 反常识 | "我穿了三十年纯棉，直到去年我放弃了" |
| 质疑 | "海藻做衣服？这不是智商税吗？" |

**规则**：钩子必须让观众产生"为什么？"的冲动。

### 2. 标杆故事（占 50-60% 篇幅）

讲一个行业标杆品牌/技术/材料的故事，用**三层递进**结构：

```
第一层：已经很极致的点
  ↓ "更夸张的是..."
第二层：更极致的点
  ↓ "然后他们还嫌不够..."
第三层：极致到离谱的点
```

**每一层必须有具体数字**（数字锤）：
- ✅ "细度只有 13 微米，你的头发大概 70 微米"
- ✅ "一只骆马只产 120 克，两年才能剪一次"
- ❌ "非常细""极其稀有"（形容词无力）

**故事素材来源**：产品文件中的"品牌故事素材"区块。

### 3. 停顿 + 感受

标杆故事讲完后，给一个**呼吸点**：

```
"这种品牌/材料/做法让我很受触动。"
"我当时没当回事，但后来一直在想这件事。"
```

**作用**：给观众消化时间，也为下一步的转折做铺垫。

### 4. 类比桥接

从标杆**自然过渡**到自己品牌/产品：

```
"我自己也在做一个品牌..."
"然后我就想了一个问题——如果这种材料连 XX 都能 XX，为什么没人把它用在 XX？"
"这就是我们图兰朵在做的事。"
```

**规则**：过渡要自然，不能突然变成广告。"类比"是关键——标杆做的事和我们做的事有什么相似的逻辑。

### 5. 谦逊收口

**降姿态但升信任**：

```
"跟 XX 比？差太远了，真的。"
"我不敢说我们能做到那样。"
"我们可能还不够好，但方向我很确定。"
```

**规则**：越谦虚，观众越信。自吹自擂会适得其反。

### 6. 轻 CTA

品牌故事型的 CTA 是**关注/关系型**，不是购买型：

```
"关注我，这种东西我会一直讲。"
"关注我，咱们慢慢聊。"
```

**禁止**：挂购物车链接、提价格、促销信息。品牌故事型的目标是建信任，不是促成交。

## 三问侧重

| 问题 | 侧重 | 实现方式 |
|------|------|---------|
| 你是谁 | ✅ 重点 | 通过讲标杆故事展现行业认知深度 |
| 为什么买 | ⚠️ 隐性 | 埋在类比桥接里，不明说 |
| 为什么便宜 | ❌ 不提 | 品牌故事型不讲价格 |
