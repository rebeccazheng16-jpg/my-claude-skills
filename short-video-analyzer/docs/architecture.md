## 流程说明

| 步骤 | 动作 | 说明 |
|------|------|------|
| ① | 用户私聊发链接 | 用户发送 TikTok/小红书链接给机器人 |
| ② | 事件订阅回调 | 飞书将消息事件 POST 到云服务器 |
| ③ | 异步处理 | 下载视频 → Gemini 分析 → 格式化结果 |
| ④ | 调用飞书 API | 发送群消息 + 写入多维表格 |
| ⑤ | 用户收到结果 | 群内收到分析卡片，Bitable 新增记录 |

## 技术选型

| 组件 | 方案 | 说明 |
|------|------|------|
| **Web 框架** | Express.js | 轻量、成熟 |
| **视频下载** | Playwright + yt-dlp | TikTok/小红书兼容 |
| **视频分析** | Gemini 2.5 Flash | 快速、便宜、多模态 |
| **任务队列** | Bull (Redis) | 可选，处理高并发 |
| **部署** | Railway / Render / VPS | 云服务器 |

## 飞书配置

### 1. 创建自建应用

1. 访问 https://open.feishu.cn/app
2. 创建企业自建应用
3. 获取 `App ID` 和 `App Secret`

### 2. 配置权限

在「权限管理」中开启：

| 权限 | 用途 |
|------|------|
| `im:message:send` | 发送消息 |
| `im:message` | 接收消息 |
| `bitable:app` | 多维表格读写 |

### 3. 配置事件订阅

在「事件订阅」中：
- 请求地址: `https://your-server.com/webhook/feishu`
- 订阅事件: `im.message.receive_v1`

### 4. 配置机器人

在「机器人」中启用机器人功能

## 环境变量

```bash
# 飞书
FEISHU_APP_ID=cli_xxxxx
FEISHU_APP_SECRET=xxxxx
FEISHU_ENCRYPT_KEY=xxxxx        # 事件订阅加密密钥
FEISHU_VERIFICATION_TOKEN=xxxxx # 事件订阅验证令牌

# 飞书群 & 表格
FEISHU_GROUP_CHAT_ID=oc_xxxxx   # 结果推送的群ID
FEISHU_BITABLE_APP_TOKEN=xxxxx  # 多维表格 App Token
FEISHU_BITABLE_TABLE_ID=xxxxx   # 数据表 ID

# Gemini
GEMINI_API_KEY=xxxxx

# 服务器
PORT=3000
```

## 多维表格结构

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 视频链接 | URL | 原始链接 |
| 平台 | 单选 | TikTok / 小红书 / 其他 |
| 分析时间 | 日期 | 自动填充 |
| 视频时长 | 数字 | 秒 |
| 类型 | 单选 | 带货/教程/娱乐/... |
| 产品 | 文本 | 产品名称 |
| 综合评分 | 数字 | 1-10 |
| 核心卖点 | 文本 | USP |
| 爆款原因 | 文本 | 分析结论 |
| 完整报告 | 附件/URL | JSON 文件链接 |
| 提交人 | 人员 | 发送链接的用户 |

## 部署方案

### 方案 A: Railway (推荐)

```bash
# 1. Fork 仓库或创建新项目
# 2. 连接 Railway
# 3. 配置环境变量
# 4. 自动部署
```

### 方案 B: VPS

```bash
# 1. SSH 到服务器
ssh user@your-server

# 2. 克隆代码
git clone https://github.com/xxx/short-video-analyzer-bot

# 3. 安装依赖
npm install
npx playwright install chromium

# 4. 配置环境变量
cp .env.example .env
vim .env

# 5. 启动服务
pm2 start server.js
```
