# 印尼语 TikTok 带货口播稿语言参考

## 一、基本语言风格原则

### 1. 口语化程度
- 使用 **bahasa gaul（印尼俚语/口语）**，不用书面语
- 省略正式语法规则，贴近日常对话
- 语气轻松、有活力、像朋友推荐
- 多用感叹词增加情绪感染力

### 2. 人称使用
| 情境 | 用法 |
|------|------|
| 称呼观众 | `guys`, `kalian`, `teman-teman`, `bestie` |
| 自称 | `aku`, `gue`（更 casual）|
| 强调亲切感 | `loh`, `dong`, `sih`, `deh`, `nih` |

### 3. 常用口语助词（Partikel Gaul）
- **loh** - 表惊讶/强调：*Ini beneran works, loh!*
- **dong** - 语气词（建议/强调）：*Cobain dong!*
- **sih** - 口语语气词：*Bagus banget sih ini!*
- **deh** - 表建议/让步：*Langsung order aja deh*
- **nih** - 表示"这个/看这个"：*Nih, aku kasih tahu rahasianya*
- **yuk** - 一起做某事：*Yuk, cobain bareng!*
- **kan** - 表确认：*Udah tahu kan?*
- **tuh** - 指示强调：*Hasilnya tuh beda banget!*

## 二、Hook（开场钩子句）模板

### 类型1：痛点共鸣型（Relatable Pain Point）
```
- "Siapa nih yang masih [masalah]? Tenang, aku nemuin solusinya!"
- "Kalian pernah nggak sih [pengalaman buruk]? Nah ini dia jawabannya!"
- "Udah coba [solusi biasa] tapi nggak mempan? Coba yang ini dulu!"
- "Buat yang [keluhan], ini wajib banget kalian coba!"
```

### 类型2：悬念反转型（Curiosity/Surprise）
```
- "Jujur ya, aku awalnya nggak percaya sama produk ini... sampai aku cobain sendiri!"
- "Nggak nyangka [hasil], padahal cuma pakai ini doang!"
- "Ini bukan iklan biasa, dengerin dulu ya!"
- "Bentar, ini serius works banget, gila sih!"
```

### 类型3：数字/结果型（Proof-based）
```
- "Dalam [X hari/minggu], [hasil]. Beneran!"
- "Udah [X ribu/juta] orang cobain, dan hasilnya..."
- "Kenapa [X orang] pada reorder terus? Karena [alasan]!"
```

### 类型4：紧迫感型（Urgency）
```
- "Eh guys, kalau kalian lagi cari [produk], stop scroll dulu!"
- "Ini produk yang lagi viral di TikTok, jangan sampai ketinggalan!"
- "Flash sale hari ini doang, simak sampai habis ya!"
```

**Hook 选择原则：**
- 美妆/护肤 → 痛点共鸣型（痘痘、暗沉、毛孔等）
- 食品零食 → 悬念/口感描述型
- 日用品/家居 → 数字/效果型
- 限时促销 → 紧迫感型

## 三、产品卖点表达方式

### 效果描述常用语
```
- "hasilnya keliatan banget dalam [waktu]"（X 时间后效果明显）
- "beda banget sama yang biasanya"（跟普通的完全不同）
- "teksturnya ringan, nggak lengket"（质地轻薄不粘腻）
- "cocok buat semua [jenis kulit/dll]"（适合所有...）
- "anti [masalah]"（不会...的问题）
- "worth it banget sama harganya"（性价比超高）
- "formula-nya udah upgraded"（配方升级版）
- "bahan-bahannya aman, udah [sertifikasi]"（成分安全，已认证）
```

### 价格/性价比表达
```
- "harganya nggak bikin kantong jebol"（价格不贵）
- "yang mahal belum tentu yang terbaik"（贵的不一定是最好的）
- "satu botol bisa [durasi pemakaian]"（一瓶能用...）
- "hemat banget kalo beli bundling"（买套餐超划算）
```

## 四、CTA（行动号召）模板

### 购物 CTA
```
- "Langsung klik keranjang kuning di bawah ya, guys!"（点击下方黄色购物车）
- "Tinggal tap keranjangnya, udah deh!"（轻点购物车就搞定）
- "Yuk order sekarang sebelum kehabisan!"（现在下单，别断货了）
- "Jangan lupa klik link-nya ya, stok terbatas!"（记得点链接，库存有限）
- "Grab it fast, sebelum promo-nya abis!"（快抢，活动快结束了）
```

### 互动 CTA
```
- "Komen 'mau' di bawah buat dapetin info lebih lanjut!"（评论"mau"获取更多信息）
- "Share ke bestie kalian yang butuh ini!"（分享给需要的好友）
- "Save dulu biar nggak lupa!"（先收藏，别忘了）
- "Follow buat konten tips selanjutnya ya!"（关注看下次的干货）
```

### 限时紧迫 CTA
```
- "Promo hari ini doang, besok harga normal!"（今天活动价，明天恢复原价）
- "Stok tinggal dikit, beneran!"（库存真的不多了）
- "Disc [X]% cuma sampai malam ini!"（今晚结束打...折）
- "Gratis ongkir hari ini, ayo buruan!"（今天免运费，快来）
```

## 五、产品类型对应的特定词汇

### 美妆/护肤（Skincare/Makeup）
```
Hook 关键词: kulit kusam, pori-pori besar, jerawat, flek hitam, kulit kering
效果词汇: glowing, cerah, mulus, lembap, kenyal, awet muda
常用词: skincare routine, double cleansing, glass skin, serum, moisturizer, SPF
```

### 食品零食（Food/Snack）
```
Hook 关键词: lapar, ngidam, cemilan enak, diet tapi mau ngemil
口感词汇: gurih, manis, renyah, kriuk-kriuk, creamy, nggak eneg
常用词: receh tapi enak, worth it, rekomendasi banget, wajib cobain
```

### 日用品/家居（Daily/Home）
```
Hook 关键词: rumah berantakan, susah bersihin, bau, nggak efisien
效果词汇: bersih maksimal, hemat waktu, anti kotor, praktis banget
常用词: life hack, game changer, wajib punya, harus ada di rumah
```

### 服装/配饰（Fashion/Accessories）
```
Hook 关键词: tampil kece, outfit of the day, OOTD, kasual tapi stylish
风格词汇: hits, kekinian, Instagrammable, aesthetic, comfy
常用词: bisa dipakai ke mana-mana, mix and match, size-nya pas
```

## 六、翻译注意事项

### 不能直译的表达（需要本地化）

| 中文表达 | ❌ 直译（避免） | ✅ 本地化（推荐） |
|----------|----------------|------------------|
| 点击购物车 | klik keranjang belanja | klik keranjang kuning di bawah |
| 限时优惠 | penawaran terbatas waktu | promo hari ini doang / flash sale |
| 效果很好 | efeknya sangat baik | works banget / hasilnya gila sih |
| 非常划算 | sangat hemat | worth it banget / nggak nyesel deh |
| 库存有限 | stok terbatas | tinggal dikit, beneran! |
| 快来购买 | segera beli | yuk order sekarang / grab it fast |
| 亲测有效 | sudah diuji pribadi | aku udah cobain sendiri, dan... |

### 语气调整原则
1. **去除过度正式的说法** - "Anda" → "kamu/kalian"
2. **增加情绪词** - 添加 gila, keren, wah, astaga 等感叹词
3. **缩短句子** - 印尼TikTok偏好短句，节奏快
4. **加入重复节奏** - 如 "beneran, beneran works!"

## 七、完整脚本结构参考

```
[HOOK - 3~5秒]
开场立即抓住注意力，提出痛点或制造好奇

[产品引出 - 5~10秒]
自然引出产品，说明"这就是解决方案"

[核心卖点 - 15~25秒]
2-3个核心卖点，配合使用场景

[社会证明（可选）- 5~10秒]
用户反馈、销量数据、个人体验

[CTA - 5~8秒]
清晰行动指引，加入紧迫感

总时长建议：30~60秒
```
