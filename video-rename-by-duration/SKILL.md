# 视频按时长重命名

## 触发条件

当用户说以下任意一种时自动触发：
- "命名 XX 文件夹"
- "更新 XX 文件夹命名"
- "重命名 XX 里的视频"
- "帮我改一下文件名"
- 任何涉及视频文件夹 + 命名/重命名的指令

## 命名规则

1. 视频文件统一按时长命名：`{秒数}s.mp4`（如 `8s.mp4`、`5s.mp4`）
2. 同文件夹内时长重复：加后缀 `8s(2).mp4`、`8s(3).mp4`，以此类推
3. 时长四舍五入取整（如 4.52s → 4s，8.06s → 8s，4.70s → 5s）
4. 保留原始扩展名（.mp4 / .mov / .MP4 均保留原格式）
5. 已符合命名规范的文件跳过不动

## 执行步骤

1. 确认目标文件夹路径
2. 用 ffmpeg 读取每个视频文件的时长
3. 按规则重命名，有重复时长自动加后缀
4. 输出重命名对照表给用户确认

## 默认监控文件夹

- `~/Desktop/可自动抓取素材/` 及其所有子文件夹
- `~/Desktop/提示词库/` 及其所有子文件夹

用户也可以指定任意其他文件夹。

## 脚本路径

`~/.claude/scripts/auto_rename_video.sh`

可通过 `bash ~/.claude/scripts/auto_rename_video.sh` 手动批量执行（扫描提示词库全部子文件夹）。
