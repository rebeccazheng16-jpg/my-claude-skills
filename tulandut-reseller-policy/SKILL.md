## 一、当前逻辑（旧版）

### 总结

| 等级 | 门槛 | 费用 | 佣金权益 |
|------|------|------|----------|
| 3星 | 2箱 | 840万 IDR ≈ 3600+ RMB | 30% 差价佣金 |
| 5星 | 10箱 | 3600万 IDR ≈ 1.5W+ RMB | 10% 额外佣金 + 10% 育成（一阶5星），按提货价计算，育成比例≈销售价×6% |
| 7星 | 10个5星 | 3.6亿 IDR ≈ 15W+ RMB | 10% 额外佣金 + 10% 育成 + 伞下全局5%提货额佣金（≈2.5%销售价） |
| 分公司 | - | - | 额外3-5%提货额佣金（≈2.5%×销售价） |

**最大佣金**：50% + 5% + 2.5% + 2.5% = **60%左右**（差价和育成不会全部同时发生）

### 旧版代理等级表

| 代理等级 | 升级规则 | 首次拿货规则 | 代理权益 |
|----------|----------|--------------|----------|
| 3星代理 | 受5星及以上代理用户邀请，注册并成为平台3星代理 | 2箱（60件/箱×2），至少拿货840万 IDR | 拿货价折扣30%（市场价7折），预估收益：一件代发市场价×30% |
| 5星代理 | 受5星及以上代理用户邀请，注册并成为平台5星代理 | 10箱（60件/箱×10），至少拿货3600万 IDR | 拿货价折扣40%（市场价6折），代理费用：直接推广10%，间接推广10%，预估收益：一件代发市场价×40%；下级3星代理囤货收益市场价×(40-30)% |
| 7星代理 | 原有：平台已招募的7星代理；新：5星代理招聘10个5星代理 | / | 拿货价折扣50%（市场价5折），代理费用：直接推广10%，间接推广10%，预估收益：一件代发市场价×40%；下级3星代理囤货收益市场价×(40-30)%，提货金额佣金：伞下所有提货金额5%的佣金 |
| 分公司代理 | 平台设置部分7星代理为分公司 | / | 拿货价折扣50%（市场价5折），代理费用：直接推广10%，间接推广10%，预估收益同上，提货金额佣金：伞下所有提货金额5%的佣金，额外佣金：伞下所有提货金额3-5%的佣金 |

## 二、新逻辑（新版）

### 核心变化

1. **加入会员单元** - 199元无商品或者含商品，需根据商品成本测算
2. **新增1星代表会员，0星代表非会员**
3. **有商品就有复购** - ToC 会更好理解，否则会比较重 ToB 再让 B 转化 C

### 新版会员等级表

| 会员等级 | 获取逻辑 | 资金成本 (IDR) | 差价 | 育成 | 培育奖励及预算 |
|----------|----------|----------------|------|------|----------------|
| 0 | 非会员 | - | - | - | - |
| 1 | 原价购买礼包2盒（6件），售价约240元人民币 | 60万 | 20% | - | - |
| 3 | 累计售出25单会员礼包 | 1500万 | 15% | 2% | 待定 |
| 5 | 横排10个3星代理 | 1.5亿 | 10% | 2% | 待定 |
| 7 | 横排5个5星 | 7.5亿 | 5% | 1% | 待定 |
| 9 | 横排3个7星 | 22.5亿 | 2.5% | - | 待定 |

**总计佣金：57.5%**

### 佣金预算

- 佣金应该还会有结余，总计可能在 **55%左右** 的整体预算
- 商品成本占比确认后，可以划分部分培育奖励及孵化预算

## 四、待讨论问题

### 问题1：会员费定价

会员费设计 **100左右** 还是 **200左右**？需要讨论剩余利润还有多少。

### 问题2：佣金变化

本质上我们会比原先发出去的佣金要多一些，因为抛开了很多不好算的逻辑，所有的差价佣金都会分出。

**等级高的一定会拿等级低的差价，无论多少层**，所以理论上对代理的诱惑更大，并且市场可以控价。

### 问题3：无实物会员卡

如果设计无实物会员卡，则佣金可以发多出去也可以齐平。

**建议**：会员卡不要跟礼包并行，因为用户在推广过程中如果有选择，实物肯定更有竞争力，尤其是 ToC 售卖。

### 问题4：晋升门槛定义

未来如果拓展其他高佣商品，包装方向应该也是类似礼包件数或者销售额。

晋升门槛可以定义为：
- **件数（人头）**
- **累计销售额**（更灵活，不然用户会对比单价差别）

**注意**：未来礼包品的综合毛利依然需要 **80%左右**

### 问题5：会员商城设计

可以设计会员商城：
- **会员**享受折扣
- **非会员**可以购买但可能并不划算

## 五、新旧对比总结

| 对比项 | 旧版 | 新版 |
|--------|------|------|
| 入门门槛 | 3星 2箱 840万 IDR | 1星 60万 IDR（礼包） |
| 等级数量 | 4级（3/5/7/分公司） | 6级（0/1/3/5/7/9） |
| 最大佣金 | ~60% | ~57.5%（预算55%） |
| 晋升逻辑 | 提货额 | 横排人数 + 销售单数 |
| ToC 友好度 | 低（重 ToB） | 高（有礼包复购） |
| 控价能力 | 弱 | 强（层层差价明确） |

## 六、快速参考

### 新版晋升路径

```
0星(非会员) → 1星(购买礼包) → 3星(卖25单) → 5星(10个3星) → 7星(5个5星) → 9星(3个7星)
```

### 差价收益链

```
9星(2.5%) ← 7星(5%) ← 5星(10%) ← 3星(15%) ← 1星(20%) ← 市场价
```

等级高的永远拿等级低的差价，无限层级穿透。
