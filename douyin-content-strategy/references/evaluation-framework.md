# 短视频内容评估框架

## 一、脚本评估体系

### 1.1 综合评分模型

```
总分 = Hook分(30%) + 完播潜力分(25%) + 互动设计分(20%) + 价值交付分(15%) + 垂直度分(10%)
```

### 1.2 各维度评分标准

#### Hook有效性（30%）

| 分数 | 标准 |
|------|------|
| 9-10 | 前3秒极具冲击力，有强烈的情绪触发或好奇心激发 |
| 7-8 | 开头有吸引力，能让大部分用户停留 |
| 5-6 | 开头一般，可能流失部分用户 |
| 3-4 | 开头平淡，"大家好"类无效开场 |
| 1-2 | 开头混乱或无关，用户大概率划走 |

**评估要点**：
- [ ] 是否在3秒内抛出吸引点
- [ ] 是否避免了无效开场词
- [ ] 是否有视觉/听觉冲击
- [ ] 是否触发情绪（好奇/共鸣/惊讶）

#### 完播潜力（25%）

| 分数 | 标准 |
|------|------|
| 9-10 | 节奏紧凑，信息密度高，无冗余，时长完美匹配内容 |
| 7-8 | 整体节奏好，偶有松散，时长基本合理 |
| 5-6 | 有些拖沓，信息密度一般 |
| 3-4 | 明显冗余，节奏松散 |
| 1-2 | 大量无效内容，严重拖沓 |

**评估要点**：
- [ ] 时长是否适合内容类型
- [ ] 每3-5秒是否有信息点变化
- [ ] 是否有结尾悬念或引导
- [ ] 是否存在可删减的冗余内容

#### 互动设计（20%）

| 分数 | 标准 |
|------|------|
| 9-10 | 多处自然的互动引导，有强烈的评论/讨论激发点 |
| 7-8 | 有明确的互动引导，能激发部分互动 |
| 5-6 | 有简单的互动引导，效果一般 |
| 3-4 | 几乎无互动设计 |
| 1-2 | 完全无互动意识 |

**评估要点**：
- [ ] 是否有评论引导语（"你觉得呢"）
- [ ] 是否有争议点/讨论点
- [ ] 是否有点赞/关注引导
- [ ] 是否设计了共鸣点

#### 价值交付（15%）

| 分数 | 标准 |
|------|------|
| 9-10 | 提供了实用/有趣/有深度的内容，用户有明确收获 |
| 7-8 | 内容有价值，但深度一般 |
| 5-6 | 有一定价值，但不够突出 |
| 3-4 | 价值模糊，用户难有收获 |
| 1-2 | 无明确价值，纯粹水内容 |

**评估要点**：
- [ ] 用户看完能获得什么
- [ ] 是否解决了某个问题/满足了某个需求
- [ ] 内容是否有独特性/差异化
- [ ] 是否值得收藏/分享

#### 垂直度（10%）

| 分数 | 标准 |
|------|------|
| 9-10 | 高度垂直，完美符合账号定位和目标受众 |
| 7-8 | 基本垂直，与账号定位一致 |
| 5-6 | 有些偏离，但关联性尚可 |
| 3-4 | 明显偏离账号定位 |
| 1-2 | 完全无关，跨领域混乱 |

## 二、数据评估体系

### 2.1 核心指标健康度

| 指标 | 差 | 一般 | 良好 | 优秀 | 顶级 |
|------|-----|------|------|------|------|
| 完播率 | <10% | 10-15% | 15-25% | 25-40% | >40% |
| 点赞率 | <0.5% | 0.5-1% | 1-3% | 3-5% | >5% |
| 评论率 | <0.05% | 0.05-0.1% | 0.1-0.5% | 0.5-1% | >1% |
| 转发率 | <0.05% | 0.05-0.1% | 0.1-0.3% | 0.3-0.5% | >0.5% |
| 收藏率 | <0.1% | 0.1-0.3% | 0.3-0.5% | 0.5-1% | >1% |
| 关注转化率 | <0.1% | 0.1-0.3% | 0.3-0.5% | 0.5-1% | >1% |

### 2.2 流量池诊断

| 播放量范围 | 可能状态 | 诊断方向 |
|-----------|----------|----------|
| <500 | 初始池未突破 | 检查完播率、账号权重 |
| 500-5K | 二级池徘徊 | 检查互动率、内容质量 |
| 5K-2W | 三级池瓶颈 | 检查转发率、粉丝转化 |
| 2W-15W | 中级池表现 | 优化内容系列化 |
| >15W | 高级池以上 | 保持质量，扩大产出 |

### 2.3 问题诊断矩阵

| 症状 | 可能原因 | 优化方向 |
|------|----------|----------|
| 完播率低+播放低 | Hook失败 | 重新设计开头 |
| 完播率高+播放低 | 标签不精准 | 检查垂直度、话题 |
| 播放高+互动低 | 内容无共鸣 | 增加情绪点、争议点 |
| 互动高+粉丝不涨 | 无关注理由 | 建立人设、系列感 |
| 数据突然下降 | 违规/限流 | 检查内容合规性 |

## 四、内容类型评估特点

### 4.1 知识干货类

**核心指标**：收藏率、完播率
**评估重点**：
- 信息价值密度
- 表达清晰度
- 实操性

### 4.2 情感故事类

**核心指标**：评论率、转发率
**评估重点**：
- 情绪共鸣度
- 故事完整性
- 真实感

### 4.3 产品种草类

**核心指标**：完播率、转化率
**评估重点**：
- 痛点挖掘准确性
- 产品展示效果
- 购买引导自然度

### 4.4 搞笑娱乐类

**核心指标**：完播率、分享率
**评估重点**：
- 笑点密度
- 创意新颖度
- 传播潜力

### 4.5 教程类

**核心指标**：收藏率、关注率
**评估重点**：
- 步骤清晰度
- 可复制性
- 内容完整性

## 五、评估检查清单汇总

### 发布前检查

- [ ] Hook是否足够吸引（3秒测试）
- [ ] 时长是否合适
- [ ] 节奏是否紧凑
- [ ] 是否有互动引导
- [ ] 是否有价值主张
- [ ] 是否垂直定位
- [ ] 标题/话题是否优化
- [ ] 封面是否吸引

### 发布后监测

- [ ] 1小时数据（初始池表现）
- [ ] 24小时数据（是否突破）
- [ ] 评论区管理（回复互动）
- [ ] 异常数据排查

### 定期复盘

- [ ] 爆款内容分析
- [ ] 低效内容诊断
- [ ] 数据趋势追踪
- [ ] 策略调整规划
