# 抖音/TikTok 内容策略知识库

## 概述

本skill提供抖音（国内）和TikTok（海外）推荐算法及内容策略的专业知识，作为知识底座支持：
- **脚本生成**：基于算法偏好优化脚本结构
- **内容分析**：评估视频/脚本的算法友好度
- **数据分析**：解读关键指标，诊断问题
- **策略优化**：制定符合平台规则的内容策略

## 核心算法原理（通用）

### 推荐公式

```
视频推荐优先级 = Σ(预测用户行为概率 × 行为价值权重)
```

两个平台的核心逻辑一致：
1. 通过神经网络预测用户对视频的行为概率（点赞、评论、分享、完播等）
2. 为不同行为赋予价值权重
3. 计算综合得分，推荐得分高的内容

### 流量池机制（通用）

```
新视频发布 → 小流量测试(200-500人) → 数据达标 → 进入更大流量池 → 循环迭代
```

**晋升逻辑**：每一级流量池的数据表现优于平均水平，即可进入下一级。

## 脚本生成指南

### 黄金结构法则

```
[0-3秒]  Hook/钩子    ——— 必须立即抓住注意力
[3-10秒] 痛点/悬念    ——— 建立共鸣或好奇心
[10-结尾] 价值交付    ——— 解决问题/提供价值/反转
[最后2秒] 行动引导    ——— 点赞/评论/关注引导
```

### 完播率优化要点

| 要点 | 说明 |
|------|------|
| 时长控制 | 抖音新手8-15秒；TikTok可尝试30-60秒 |
| 开头禁忌 | 避免"大家好"等无效开场 |
| 信息密度 | 每3-5秒一个信息点或视觉变化 |
| 结尾设计 | 设置悬念、引导二次观看或互动 |

### Hook公式模板

| 类型 | 公式 | 中文示例 | 英文示例 |
|------|------|----------|----------|
| 数字型 | "X个方法/技巧" | "3个让你涨粉10倍的技巧" | "3 tricks to 10x your followers" |
| 痛点型 | "你是不是也..." | "你是不是也总被老板挑刺？" | "Are you also tired of..." |
| 反常识 | "其实X是错的" | "其实喝8杯水是错的" | "Everything you know about X is wrong" |
| 悬念型 | "直到X，我才发现" | "直到亏了10万，我才发现这个坑" | "I didn't realize until..." |
| 利益型 | "学会这个，你能..." | "学会这个，月入过万" | "This one thing changed my..." |

### 内容类型时长建议

| 内容类型 | 抖音 | TikTok | 关键指标侧重 |
|----------|------|--------|--------------|
| 知识干货 | 30-60秒 | 60-90秒 | 收藏率、完播率 |
| 情感故事 | 15-30秒 | 30-60秒 | 评论率、分享率 |
| 产品种草 | 15-20秒 | 15-30秒 | 完播率、转化率 |
| 搞笑娱乐 | 8-15秒 | 15-30秒 | 完播率、分享率 |

## 内容评估标准

### 脚本质量评分

| 维度 | 权重 | 评估要点 |
|------|------|----------|
| Hook有效性 | 30% | 前3秒是否有足够吸引力 |
| 完播潜力 | 25% | 节奏、时长、信息密度 |
| 互动设计 | 20% | 是否有评论/点赞引导点 |
| 价值交付 | 15% | 内容是否有用/有趣/有共鸣 |
| 垂直度 | 10% | 是否符合账号定位 |

### 评估检查清单

生成或分析脚本时，检查以下要点：

- [ ] 前3秒是否有强Hook
- [ ] 时长是否适合内容类型
- [ ] 是否有明确的价值主张
- [ ] 节奏是否紧凑（无冗余）
- [ ] 是否设计了互动点
- [ ] 结尾是否有行动引导
- [ ] 是否符合账号垂直领域

## 数据诊断指南

### 问题诊断决策树

```
播放量低？
├── 完播率<15% → Hook不够吸引，优化开头
├── 完播率正常但播放低 → 标签不精准，检查垂直度
└── 完播率高但播放低 → 互动率不足，增加互动引导

有播放无互动？
├── 内容太"平" → 增加情绪点/争议性/共鸣点
├── 无互动引导 → 加入"你觉得呢"等引导语
└── 与用户无关 → 检查目标受众匹配度

粉丝不涨？
├── 无"关注理由" → 建立期待感，预告下期
├── 人设不清晰 → 强化差异化定位
└── 无系列感 → 建立内容IP或系列专题
```

### 数据健康参考值

| 指标 | 差 | 一般 | 良好 | 优秀 |
|------|-----|------|------|------|
| 完播率 | <10% | 10-20% | 20-40% | >40% |
| 点赞率 | <1% | 1-3% | 3-5% | >5% |

| 转发率 | <0.1% | 0.1-0.3% | 0.3-0.5% | >0.5% |

## 2025年算法趋势

### 抖音
- 去超头化：扶持中小创作者和品牌自播
- 搜索推荐联动：搜索词相关内容获得更多分发
- 长期价值导向：关注用户长期行为而非仅即时互动
- 算法透明化：官方公开算法原理

### TikTok
- 长视频偏好：>30秒视频获得更多推荐
- 收藏/分享权重提升：强指标信号
- 社群机制：基于共同兴趣的用户分组
- 内容质量优先：原创性、制作质量影响推荐
