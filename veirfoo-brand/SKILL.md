## 两大核心科研成分

### 1. 500D超微小分子水解胶原蛋白（核心专利）
- 分子量：500道尔顿以下
- 尺寸：相当于毛孔的九亿九千七百万分之一
- 机理：可直接穿过皮肤毛孔，透皮吸收，实现**胶原直补**
- 灵感来源：医美面部填充项目（玻尿酸→胶原蛋白填充）
- 意义：让面膜具备医美充盈效果，实质性胶原抗老功效

### 2. PDRN（多聚脱氧核糖核苷酸 / DNA钠）
- 来源：三文鱼生殖细胞中提取的特定规格DNA片段
- 分子量：50～1500kDa之间
- 与人类DNA碱基相似度：**98%**，无免疫原性，无毒无副作用
- 历史：1952年意大利Mastelli公司率先提取研究
- 平均分子量：≤1000KD，含量≥5000ppm

**PDRN四大功效：**
1. 与腺苷A2A受体结合 → 打开细胞通道，抗炎
2. 促进EGF、FGF、IGF等生长因子分泌 → 重塑受损皮肤内环境
3. 促进VEGF生成毛细血管 → 为皮肤供给营养
4. 通过补救途径加速DNA合成 → 皮肤迅速再生
