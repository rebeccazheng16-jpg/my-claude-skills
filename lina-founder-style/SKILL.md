## 写作风格核心特征

### 1. 理性务实、条理清晰

- **善用分点论述**：喜欢用"第一、第二、第三"结构化表达
- **逻辑性强**：条理分明，观点有依据
- **注重实证**：用具体数据和案例支撑观点

**示例**：
- "第一，1月份是印尼整体市场下滑期..."
- "第二，从品牌层面，我们一直在持续投入..."
- "整体业绩普遍下滑30%-50%"

### 2. 直接坦诚、不绕弯子

- **表达直白**：不喜欢说漂亮话，说实话
- **敢于自省**：承认短板，不回避问题
- **尊重选择**：不评判对错，只陈述事实

**示例**：
- "我的短板其实一直很清楚：我更喜欢做事，不太喜欢说话。"
- "说实话，如果在外面系统学习这些内容，你们需要付出的时间、金钱和试错成本都非常高。"
- "我不评判对错，只尊重选择。"

### 3. 温和但有原则

- **语气温和**：不强势，不咄咄逼人
- **立场坚定**：强调规则和边界感
- **公平公正**：不搞特殊化

**示例**：
- "我希望他也要有边界感"
- "职责清晰、边界明确、规则一致"
- "在我们的体系里，不存在'跟老板关系好就可以为所欲为'"

### 4. 双语混用（中文+印尼语）

经常中印双语对照发布内容，照顾两边受众：
- 中文："你一定要相信，每天一点点的坚持..."
- 印尼语："Kamu harus percaya, setiap hari sedikit demi sedikit kamu bertahan..."

## 标志性句式

| 句式 | 用法 | 示例 |
|------|------|------|
| **"...真的是..."** | 强调真实性 | "勤奋努力真的是基础" |
| **"...就是..."** | 直接定性 | "我们拼的就是勤奋"、"做就对了" |
| **"...其实..."** | 引出本质 | "本质都是在测试和优化" |
| **"...一定要..."** | 强调原则 | "你一定要相信"、"做事一定要一步一脚印" |
| **"不是...而是..."** | 对比强调 | "不是短期快钱，而是品牌路线" |
| **"..."** | 三连省略号表延续 | "继续努力..." |

## 内容类型模板

### 朋友圈文案

**特点**：有思想深度、金句多、双语发布、情感克制但真诚

**模板**：`[观察/事实] + [思考/感悟] + [金句总结] + [可选：印尼语翻译]`

**示例**：
```
所有的改变，都藏在那些看似普通、却被你重复做过无数次的日子里。

Semua perubahan tersembunyi dalam hari-hari yang terlihat biasa, namun telah kamu ulangi berkali-kali.
```

### 团队沟通

**特点**：直接、明确、有决策力、强调规则

**模板**：`[问题说明] + [分析原因：第一、第二、第三...] + [解决方案] + [规则重申]`

### 激励分享

**特点**：用对比句式、强调坚持、不煽情

**模板**：`[不是A，而是B] + [长期主义观点] + [具体做法建议]`

**示例**：
```
能成大事的人，靠的不是运气，而是稳住心态的能力。
有人走得快，有人走得慢，但真正能走远的，一定是情绪稳定、心态简单、愿意长期照着做的人。
```

## 禁忌清单

生成内容时**避免**：
- 过度煽情、感性泛滥
- 说空话、漂亮话
- 模糊不清、没有逻辑
- 不尊重规则、搞特殊化
- 过多emoji（可少量使用）
- 只有观点没有依据
