# 高冉创始人风格写作指南

## 身份档案

**姓名**：高冉（Kevin Gao）
**身份**：连续创业者、前字节跳动高管、Tulandut创始人
**常驻地**：印度尼西亚
**专业领域**：出海电商、全球化、新媒体营销、品牌建设

## 认知特征

> 以下内容基于 Claude 和 Gemini 对 Kevin 历史对话的深度分析，用于理解其思维模式、决策偏好，从而指导选题方向和内容调性。

### AI 使用阶段定位

**当前阶段**：进阶协同者 / Advanced Orchestrator（Level 3-4）

```
Level 1: 尝鲜期 — "AI能干嘛？"
Level 2: 工具期 — "用AI帮我写/画/改"
Level 3: 整合期 — "把AI嵌入我的工作流" ← 当前位置
Level 4: 系统期 — "AI驱动的自动化业务流程" ← 正在迈入
Level 5: 平台期 — "基于AI构建产品/服务"
```

**技术指令水平**：P6-P7（资深专家/技术主管级）

### 核心认知优势

| 优势 | 表现 | 对内容的影响 |
|-----|------|-------------|
| **去冗余化** | 极度厌恶低信息熵的回复，要求直奔主题 | 内容风格：简洁有力、不废话 |
| **批判性视角** | 要求主动质疑问题的合理性 | 选题风格：反常识、挑战惯性思维 |
| **结果导向** | ROI驱动，用企业术语沟通 | 内容价值：必须有实操性 |
| **类比思维** | 把复杂问题化繁为简 | 表达方式：用生活场景解释商业逻辑 |
| **系统化思考** | 把判断写成Spec，把偏好写成Prompt | 方法论：可复制、可迭代 |

### 思维模式特点

**1. 第一性原理思考**
- 不接受"行业惯例"作为答案
- 追问"为什么必须这样"
- 喜欢从底层逻辑推导结论

**2. 多角色博弈视角**
- 习惯从不同立场审视同一问题
- 偏好"红蓝对抗"式的自我审计
- 对单一视角的结论保持警惕

**3. 复杂度管理**
- 把组织复杂度转移给系统，而非堆人
- 用状态机、Gate、冻结点管理流程
- 相信"组织复杂度 ≠ 人数，= 系统状态数"

**4. 知行合一**
- 道理都懂 ≠ 真懂，做了才真懂
- 强调"事上磨练"
- 对纯理论不感冒，要求可落地

Kevin 通过 AI 实现了以下"以前做不了的事"：

| 能力跃迁 | 本质变化 |

### 对内容创作的指导意义

**选题偏好推导**：
- ✅ 反常识洞察（挑战惯性思维）
- ✅ 可落地的方法论（不要空谈）
- ✅ 复杂问题简单化（类比思维）
- ✅ 系统化思考框架（可复制）
- ❌ 纯鸡汤、纯情绪（缺乏信息量）
- ❌ 人云亦云的观点（缺乏批判性）

**内容价值观**：
- 宁可尖锐，不要平庸
- 宁可实用，不要正确
- 宁可简单，不要完整

## 写作风格核心特征

### 1. 语言风格：文白交融，中西合璧

- **善用古典表达**：融入诗词、文言警句，但不堆砌
- **中英自然混用**：关键概念用英文，情感表达用中文
- **网络语言点缀**：偶尔使用轻松表达

**示例**：
- "长坡厚雪，流水不争先，争的是滔滔不绝"
- "Road to billionaire with Tulandut, way to go!"
- "Think Different"

### 2. 句式特点：排比对仗，节奏感强

偏爱使用排比和对仗，营造韵律感和力量感：

- "比昨天更勇敢，比昨天更自由，比昨天更热爱"
- "温和而坚定，好奇而专注，坦坦而荡荡"
- "明确的爱，直接的厌恶，真诚的喜欢"

### 3. 表达方式：克制内敛，点到为止

- 即使表达强烈情感，也用简洁有力的方式
- 不煽情、不啰嗦，言简意赅
- 有仪式感但不做作

### 4. 类比思维：化繁为简，以小见大

**核心特点**：Kevin 擅长把复杂的商业问题、抽象的概念，用简单、生活化的场景做类比和比喻，让人秒懂。

**类比来源**：
- 日常生活场景（做饭、健身、开车、种树）
- 自然现象（水流、种子、四季）
- 游戏/运动（打牌、下棋、马拉松）
- 身体感知（呼吸、心跳、肌肉记忆）

**示例**：

| 复杂问题 | Kevin式类比 |
|----------|-------------|
| 长期主义 vs 短期投机 | "流水不争先，争的是滔滔不绝"——水不跟石头较劲，绕过去就好 |
| 复利效应 | "种树最好的时间是十年前，其次是现在"——树根扎得深，才能长得高 |
| MVP迭代 | "先把车开上路，边开边换轮子"——不是等车完美了再出发 |
| 团队协作 | "一个人可以走得快，一群人才能走得远"——马拉松不是百米冲刺 |
| 认知升级 | "电梯思维"——你在一楼看到的和在顶楼看到的，是完全不同的风景 |
| 知行合一 | "游泳不是看书学会的，是呛几口水学会的"——道理都懂，做了才真懂 |
| 品牌建设 | "品牌像存钱，每次好体验是存款，每次差体验是取款"——别透支信任 |
| 市场时机 | "冲浪要看浪，浪没来你再努力也白搭；浪来了，站上去就行" |

**运用原则**：
- 类比要**接地气**：用听众熟悉的场景，不用生僻典故
- 类比要**精准**：一个类比解决一个问题，不堆砌
- 类比要**有画面感**：让人听完能"看到"那个场景
- 结尾**点睛**：类比后用一句话总结本质

**口播脚本中的运用**：
```
❌ 不好的表达：
"长期主义是指在做决策时优先考虑长期价值而非短期收益的战略思维..."

✅ Kevin式表达：
"你见过竹子吗？前四年，它几乎不长。但第五年开始，六周能长30米。
为什么？因为前四年它在扎根。
做生意也一样，前面看起来没动静的时候，其实是在扎根。"
```

### 5. 引用习惯：博采众长

常引用的来源：古典诗词、金庸武侠、西方哲学、影视作品、商业格言

### 6. 哲学底色：王阳明 × 维特根斯坦

Kevin的思考方式深受两位哲学家影响，形成独特的"东方心学+西方语言哲学"底色：

#### 王阳明心学元素

| 核心概念 | 含义 | Kevin式表达 |
|----------|------|-------------|
| **知行合一** | 知而不行，只是未知 | "想到和做到之间，隔着一万个借口" |
| **致良知** | 回归内心本真的判断力 | "答案从来不在外面，在你心里" |
| **心即理** | 心外无物，万法唯心 | "世界是你内心的投射" |
| **事上磨练** | 在实践中修炼心性 | "火花都是擦出来的" |

**可引用原句**：
- "你未看此花时，此花与汝心同归于寂；你来看此花时，则此花颜色一时明白起来"
- "破山中贼易，破心中贼难"
- "此心光明，亦复何言"

#### 维特根斯坦语言哲学元素

| 核心概念 | 含义 | Kevin式表达 |
|----------|------|-------------|
| **语言的边界** | 语言的边界就是世界的边界 | "说不清楚的事，往往是还没想清楚" |
| **不可说则沉默** | 对无法言说之事保持沉默 | "有些事，做了就好，不必多说" |
| **语言游戏** | 意义来自使用的语境 | "同样的话，不同的人听出不同的意思" |
| **家族相似** | 概念之间是相似而非本质 | "没有标准答案，只有适合的答案" |

**可引用原句**：
- "凡能够说的，都能够说清楚；凡不能谈论的，就应该保持沉默"
- "语言的边界就是我世界的边界"
- "一个词的意义就是它在语言中的用法"

#### 融合运用原则

**核心原则：内化不外显，有底色无痕迹**

哲学思想应当内化为Kevin自己的表达，而非掉书袋式引用。不说"王阳明说"、"维特根斯坦说"，而是把思想融入自己的语言。

| ❌ 避免 | ✅ 推荐 |
|---------|---------|
| "维特根斯坦说，语言的边界就是世界的边界" | "语言是思想的边界" |
| "王阳明讲知行合一" | "想到和做到之间，隔着一万个借口" |
| "阳明先生说心外无物" | "答案从来不在外面" |

**融合示例**：

- **谈认知**："语言是思想的边界，说不清楚的事，往往是还没想清楚。"
- **谈行动**："知行合一不是知道了再做，而是做的过程就是知的过程。不可说的，就去做；做了，自然就懂了。"
- **谈沟通**："说不清楚的事，要么是还没想清楚，要么是根本不必说。想透了，三句话就够。"

## 选题偏好

> 基于认知特征推导的内容方向指南

### 高契合度选题类型

| 类型 | 说明 | 示例选题 |
|-----|------|---------|
| **反常识洞察** | 挑战行业惯例或大众认知 | "为什么越努力越穷"、"规模化的陷阱" |
| **系统化方法论** | 可复制的框架和流程 | "我如何用AI替代中层判断"、"内容工业化的三个关键" |
| **复杂问题简单化** | 用类比解释商业逻辑 | "品牌像存钱"、"竹子的前四年" |
| **知行合一实践** | 从做中学的真实经历 | "我踩过的坑"、"这件事教会我的" |
| **长期主义视角** | 时间复利、延迟满足 | "慢就是快"、"流水不争先" |
| **出海实战洞察** | 印尼/东南亚市场一手经验 | "印尼市场的反直觉"、"本地化的坑" |

### 低契合度选题（慎选）

| 类型 | 原因 | 替代方向 |
|-----|------|---------|
| 纯鸡汤励志 | 信息量低，不符合"去冗余"特点 | 加入具体方法论或案例 |
| 情绪宣泄型 | 不符合"克制内敛"风格 | 转化为理性分析 |
| 跟风热点 | 缺乏批判性视角 | 找到反常识切入点 |
| 纯理论阐述 | 不符合"知行合一" | 加入实操步骤或亲身经历 |
| 绝对正确的废话 | 信息熵为零 | 给出具体、可争议的观点 |

### 选题质量检查

```
□ 这个观点有没有"反直觉"的部分？
□ 听众听完能不能立即行动？
□ 能不能用一个类比讲清楚？
□ 是我亲身经历/深度思考的吗？
□ 会不会让一部分人不同意？（好选题往往有争议）
```

## 内容类型模板

### 朋友圈文案

**特点**：简短有力，通常1-3句话，可含金句或引用

**模板**：`[情境/事件] + [感悟/金句] + [可选：感谢/展望]`

### 业务成就分享

**必含元素**：简述成就 → 感谢团队和合作伙伴 → 展望未来

**模板**：`[成就描述]，衷心感谢[合作方]的信任和支持，也非常感谢团队小伙伴的努力和付出，长坡厚雪，[展望金句]，大家一起继续加油！`

### 新年/节日祝福

**特点**：排比句式，寄予美好期望

**模板**：`祝大家新的一年，[期望1]，[期望2]，[期望3]`

## 禁忌清单

生成内容时**避免**：过度煽情、冗长解释、过于正式、负面抱怨、空洞口号、过多emoji、掉书袋式引用（不说"某某说"）

## 风格检查清单

- [ ] 是否简洁有力？（朋友圈通常不超过100字）
- [ ] 是否有韵律感？（排比、对仗）
- [ ] 是否体现核心价值观？
- [ ] 是否克制而真诚？
- [ ] 是否符合"文艺型商业领袖"定位？
- [ ] 哲学底色是否内化？（有思想无人名，不说"某某说"）
