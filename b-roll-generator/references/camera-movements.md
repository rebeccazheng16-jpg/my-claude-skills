# 镜头语言参考库

用于 AI 视频生成的镜头运动和视觉风格术语。

## 镜头景别 (Shot Sizes)

| 中文 | English | 适用场景 |
|-----|---------|---------|
| 远景/全景 | Wide shot / Establishing shot | 环境展示、场景建立 |
| 中景 | Medium shot | 人物动作、日常场景 |
| 近景 | Close-up | 细节强调、情感表达 |
| 特写 | Extreme close-up | 微观细节、强烈情感 |
| 过肩镜头 | Over-the-shoulder shot | 对话、交互场景 |

## 镜头运动 (Camera Movements)

| 中文 | English | 效果描述 |
|-----|---------|---------|
| 推镜头 | Dolly in / Push in | 聚焦、强调、进入情境 |
| 拉镜头 | Dolly out / Pull out | 揭示全貌、退出情境 |
| 横移 | Tracking shot / Truck | 跟随动作、展示空间 |
| 摇镜头 | Pan (horizontal) | 环顾环境、连接元素 |
| 升降 | Tilt (vertical) | 揭示高度、展示规模 |
| 航拍 | Aerial shot / Drone shot | 宏观视角、壮观场景 |
| 环绕 | Orbit / Arc shot | 360度展示、戏剧化效果 |
| 手持 | Handheld | 紧张感、真实感、临场感 |
| 稳定器 | Gimbal / Steadicam | 流畅跟随、专业质感 |

## 速度与时间 (Speed & Time)

| 中文 | English | 适用场景 |
|-----|---------|---------|
| 慢动作 | Slow motion | 情感渲染、细节展示 |
| 快动作 | Fast motion / Speed ramp | 紧迫感、时间压缩 |
| 延时摄影 | Time-lapse | 时间流逝、变化过程 |
| 定格 | Freeze frame | 强调瞬间、暂停思考 |
| 超级慢动作 | Hyperlapse | 城市穿梭、大范围时间流逝 |

## 常用组合模板

### 商业/创业内容
```
[主体] in modern office, professional lighting, shallow depth of field, warm corporate tones, slow dolly in
```

### 科技/产品展示
```
[主体] with holographic interface, futuristic blue lighting, sleek minimal background, smooth orbit shot
```

### 情感/故事叙述
```
[主体] in [场景], cinematic lighting, film grain, golden hour, handheld camera movement
```

### 数据/分析内容
```
Data visualization, animated charts, glowing numbers, dark background with blue accents, smooth transitions
```

### 自然/生活方式
```
[主体] in natural setting, soft natural lighting, warm earth tones, gentle camera movement, peaceful atmosphere
```
