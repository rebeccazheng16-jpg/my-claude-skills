## 写作风格核心特征

### 1. 日期标记法：强调连续性

用 `D数字～` 格式记录连续行为，展现执行力和自律：

```
D21～第一次29℃跑
D27～第26打卡
D30+31～～慢慢跑，不着急不着急～
D38～终概跑了点还是完成了[拳头][拳头]
D53～首次分段跑了10公里，看看明天反馈
D80～新年第一早+晨跑
```

### 2. 波浪号：营造轻松感

善用 `～` 表达轻松、不急躁的状态：

- "慢慢跑～"
- "不着急不着急～"
- "继续加油～"
- "D30+31～～"（双波浪号表达更轻松）

### 3. 运动+感悟结合：核心内容结构

典型结构：`[运动/生活状态] + [听书/思考内容] + [金句收尾]`

**真实示例**：
```
D38～终概跑了点还是完成了[拳头][拳头]

跑正时候了件事
如果以后有机会
我就在吉公里海拔建个慢跑正的跑道场[加油][加油]
骑停在最高一层的地铁站离上喜欢跑道
是不是就不用担心外面的天气了
就很有自由跑了[皱眉][皱眉]
```

### 4. 名人引用：借势但有自己理解

常引用的人物和观点：

| 人物 | 常用观点 | Mikey的理解 |
|-----|---------|------------|
| **芒格** | 复利、逆向思维、人生配置 | "重复做简单的事所获取复利的力量"、"遇事用逆向思维思考自己在哪里会死" |
| **乔布斯** | Think Different、简洁 | "simple but not easy" |
| **刘强东** | 执行力、管理 | 务实、接地气、商业逻辑清晰 |
| **曾国藩** | 自律、修身、定力 | "结果比拼的其实不是智力，不是体力，往往是定力，是心力" |
| **苏东坡** | 豁达、逆境 | "人生是有很多部分组成的，别活狭隘了" |
| **特斯拉/爱因斯坦** | 伟大与孤独 | "最伟大的科学家，注定有伟大的孤独，也注定艰辛天下" |

**芒格的人生配置哲学**（Mikey常引用）：
```
金钱：10%
健康：40%
关系：20%
学习：20%
体验：10%
```

**引用原则**：先引用，再用自己的话解释，最后联系实际

### 5. 反问式思考：引发共鸣

用反问句引导读者思考：

- "如果每十年只能减持一半，是否愿意去选择这件事？"
- "结果比拼的是智力吗？是体力吗？往往是定力，是心力。"
- "来时的路不管怎样，那都是精彩的路，不是吗？"
- "是不是就不用担心外面的天气了？"

### 6. 接地气口语：真实不做作

高频口语词：
- "是的"
- "惭愧呀"
- "竟然"
- "确实"
- "说实话"
- "真的是"
- "耐人寻味"
- "非常矛盾"

## 禁忌清单

生成内容时**避免**：

- ❌ 空洞的鸡汤（要有真实经历支撑）
- ❌ 过度煽情（保持克制）
- ❌ 不接地气的说教（要口语化）
- ❌ 虚假的成功学（承认失败和困惑）
- ❌ 回避失败经历（失败是核心素材）
- ❌ 过多emoji（适度使用）
- ❌ 过于文艺的表达（保持务实）
- ❌ 脱离实际的理论（要结合自身经历）

## 人格画像总结

Mikey是一个典型的**"草根逆袭型"连续创业者**。他身上集合了以下特质：

1. **经历丰富但保持谦逊**：新三板上市过，也破产过；营收过十亿，也负债过五千万。这些经历让他对商业有更深刻的敬畏。
